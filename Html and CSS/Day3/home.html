Navbar Theory

Why use display: inline-block?
inline-block lets elements sit next to each other (like inline) but still allows setting width, height, and margins (like block).

Difference between inline, block, and inline-block:
- inline: Flows with text, cannot set width/height, margins only work horizontally.
- block: Starts on a new line, takes full width by default, can set width/height.
- inline-block: Sits inline but can have width/height/margins like a block.

Sticky Navbar Example

Fixed vs Sticky:

Fixed: Stays at the same position on the screen no matter where you scroll.
Sticky: Acts like relative until the scroll reaches its top position, then it "sticks" until its parent element is out of view.
